# This is NOT the Next.js you know

This version has breaking changes — APIs, conventions, and file structure may all differ from your training data. Read the relevant guide in `node_modules/next/dist/docs/` before writing any code. Heed deprecation notices.

---

## Agent Guidelines

When working on Tadnun, all agents must:

1. **Respect trilingual content** — changes to UI must work in FR, EN, and AR (RTL)
2. **Check Next.js 16 docs** — never rely on training data for Next.js APIs
3. **Keep files small** — max 200 lines for components, split when exceeded
4. **Conventional commits** — all commits use `feat:`, `fix:`, `refactor:`, etc.
5. **Sectors are extensible** — never hardcode the sector list as final
6. **Ask before acting** — check with the user before significant decisions
